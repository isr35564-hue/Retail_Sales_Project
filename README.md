## Folder Description

### code

Contains Python scripts used for data profiling, cleaning, transformation, and KPI generation.

### output

Contains cleaned datasets and KPI output files generated by the pipeline.

### powerbi

Contains the Power BI dashboard file.

### documentation

Contains the project report, architecture diagram, and dashboard screenshots.


## Key Insights

* Electronics generated the highest revenue among all product categories.
* Laptop was the highest revenue-generating product.
* Chennai recorded the highest city-level revenue.
* Revenue distribution across major cities was relatively balanced.
* Clothing contributed the lowest share of total revenue.
* Payment methods showed a fairly balanced distribution across transactions.
* Data standardization improved reporting accuracy by eliminating product and category naming inconsistencies.
